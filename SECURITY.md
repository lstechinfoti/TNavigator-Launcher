# Política de Segurança

## Versões suportadas

| Versão | Suporte |
|---|---|
| 3.0.0 | Sim |

---

## Downloads oficiais

Baixe somente pela página oficial de releases:

```text
https://github.com/lstechinfoti/TNavigator-Launcher/releases/latest
```

Não execute instaladores baixados de fontes desconhecidas.

---

## Verificação SHA256

Cada release deve possuir:

```text
TNavigatorLauncher_Setup_3.0.0.exe
SHA256.txt
```

Hash esperado da versão atual:

```text
3F2B30DE919D3FB1382C102C8CBBB6C7163D6143E55E9D9FC0AA73A5CC35D0E2
```

Para verificar:

```powershell
Get-FileHash .\TNavigatorLauncher_Setup_3.0.0.exe -Algorithm SHA256
```

Compare o resultado com o arquivo `SHA256.txt` da release.

---

## Como reportar problemas de segurança

Se encontrar algum problema relacionado à segurança, envie ao mantenedor:

- Versão do launcher.
- Versão do Windows.
- Descrição do problema.
- Passos para reproduzir.
- Logs, se possível.
- Print do erro, se existir.
