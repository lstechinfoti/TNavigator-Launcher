# ⚡ TNavigator Launcher 3.0.0

🚀 Chegou a versão pública do **TNavigator Launcher 3.0.0**.

Um launcher moderno, visualmente premium e criado para entregar uma experiência mais rápida, organizada e segura na instalação, atualização, backup e manutenção do TNavigator no **Windows 10/11**.

---

## 🔥 Novidades principais

- 🎮 Visual moderno com identidade premium
- ⚡ Splash screen personalizada
- 🛡️ Validação obrigatória com **SHA256**
- 📦 Instalação e atualização do TNavigator
- 🧩 Verificação de dependências **Microsoft VCLibs**
- 🔐 Gerenciamento de certificado
- ☁️ Backup local e OneDrive
- ♻️ Restauração de backup `.zip`
- 📋 Logs detalhados com filtros
- 🧠 Tela Sobre com diagnóstico técnico
- 🖥️ Layout responsivo para notebook 1366x768
- 🌐 Integração com GitHub Releases/version.json

---

## 📥 Download

Baixe o instalador oficial:

```text
TNavigatorLauncher_Setup_3.0.0.exe
```

Também estão disponíveis:

```text
TnavigatorLauncher3_3.0.0.zip
SHA256.txt
```

Use o `.exe` para instalação profissional no Windows. Use o `.zip` para versão compilada/manual.

## 🧪 Conferência de segurança

Use o PowerShell para verificar o SHA256:

```powershell
Get-FileHash .\TNavigatorLauncher_Setup_3.0.0.exe -Algorithm SHA256
```

O resultado deve ser igual ao hash informado no arquivo `SHA256.txt`.
