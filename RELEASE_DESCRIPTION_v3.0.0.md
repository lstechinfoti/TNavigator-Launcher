# TNavigator Launcher 3.0.0

🚀 Versão pública do **TNavigator Launcher 3.0.0**.

O TNavigator Launcher é um instalador, atualizador, gerenciador de backup, restauração, logs e diagnóstico para Windows 10/11.

## Destaques

- Interface moderna e organizada.
- Splash screen premium.
- Instalador profissional para Windows.
- Backup local e OneDrive.
- Restauração de backup `.zip`.
- Logs com filtros.
- Validação SHA256 obrigatória.
- Diagnóstico técnico na tela Sobre.
- Temas visuais.
- Abertura maximizada por padrão.

## Download

Baixe o arquivo:

```text
TNavigatorLauncher_Setup_3.0.0.exe
```

## Requisitos

- Windows 10/11.
- Arquitetura x64 recomendada.
- Permissão de administrador para operações protegidas.
