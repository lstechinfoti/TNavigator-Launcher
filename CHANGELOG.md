# Changelog

Todas as mudanças importantes do projeto serão registradas aqui.

## [3.0.0] - 2026-05-02

### Adicionado

- Primeira versão pública do TNavigator Launcher
- Interface moderna em WPF
- Instalação e atualização do TNavigator
- Validação SHA256
- Instalação de Microsoft VCLibs
- Instalação e remoção de certificado
- Backup e restauração do LocalState
- Logs com filtros
- Tela Sobre com diagnóstico técnico
- Splash screen personalizada
- Site oficial em GitHub Pages

### Corrigido

- Ajustes de responsividade para 1366x768
- Ajustes em dialogs responsivos
- Correção do botão Abrir TNavigator para não abrir o launcher novamente
- Correção de certificado MSIX 0x800B0109
