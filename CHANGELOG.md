# Changelog

Todas as mudanças importantes do **TNavigator Launcher** serão documentadas aqui.

---

## [3.0.0] - 2026-05-02

### Adicionado

- Interface moderna do TNavigator Launcher 3.0.
- Instalador profissional para Windows.
- Splash screen premium.
- Abertura maximizada por padrão.
- Instalação e atualização do TNavigator.
- Validação obrigatória por SHA256.
- Verificação de dependência Microsoft VCLibs.
- Instalação e remoção de certificado.
- Backup local e OneDrive.
- Restauração de backup `.zip`.
- Tela de logs com filtros.
- Exportação/cópia de logs.
- Tela Sobre com diagnóstico técnico.
- Temas visuais e personalização.
- Dialogs modernos de confirmação, sucesso e aviso.
- Distribuição via GitHub Releases.

### Melhorado

- Organização visual das telas.
- Experiência do usuário durante instalação e atualização.
- Layout da tela de Backup.
- Layout da tela de Instalação.
- Layout da tela de Logs.
- Layout das telas Sobre e Créditos.
- Estabilidade do SplashWindow.
- Empacotamento com Inno Setup.

### Segurança

- SHA256 obrigatório para arquivos críticos.
- Diagnóstico técnico para suporte.
- Separação de logs por data.
- Certificado controlado com segurança.

### Distribuição

- Release oficial via GitHub Releases.
- Instalador `.exe` anexado na release.
- Arquivo `SHA256.txt` anexado para verificação.
