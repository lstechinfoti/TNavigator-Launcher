# Guia rápido para postar no GitHub

Repositório:

```text
https://github.com/lstechinfoti/TNavigator-Launcher
```

## Arquivos para enviar no repositório

Envie todos os arquivos deste ZIP para a raiz do repositório.

Principais:

```text
README.md
INSTALL.md
CHANGELOG.md
SECURITY.md
SUPPORT.md
version.json
docs/
```

## Release

Na release `v3.0.0`, anexe:

```text
TNavigatorLauncher_Setup_3.0.0.exe
SHA256.txt
```

## Descrição da release

Use o conteúdo do arquivo:

```text
RELEASE_DESCRIPTION_v3.0.0.md
```

## Conferir depois de postar

- O README mostra os prints.
- O botão de download abre a release.
- O link direto do instalador funciona.
- O version.json abre pelo raw GitHub.
