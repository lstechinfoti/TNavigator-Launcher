# Suporte

Para solicitar suporte do **TNavigator Launcher**, envie as informações abaixo.

---

## Informações necessárias

- Versão do launcher.
- Versão do Windows.
- Print do erro.
- Log exportado.
- Diagnóstico da tela **Sobre**.
- Descrição do que estava fazendo quando o erro aconteceu.

---

## Onde encontrar os logs

No launcher:

```text
Menu lateral > Logs
```

Você pode usar:

- Copiar logs.
- Exportar logs.
- Abrir pasta.

---

## Onde copiar diagnóstico

No launcher:

```text
Menu lateral > Sobre
```

Clique em:

```text
Copiar diagnóstico
```

---

## Problemas comuns

### Erro ao baixar arquivo

Verifique:

- Internet.
- Link do manifesto.
- Link do arquivo.
- Permissão do Windows.
- Antivírus/firewall.

### Erro de SHA256

Isso pode indicar:

- Hash incorreto no manifesto.
- Arquivo alterado.
- Download incompleto.
- Arquivo antigo em cache.

### Erro com certificado

Tente:

- Executar como administrador.
- Conferir logs.
- Remover certificado pelo launcher.
- Instalar novamente.

### Backup pequeno ou sem LocalState

Verifique se o TNavigator gerou dados locais antes do backup.
