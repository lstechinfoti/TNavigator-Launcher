# Instalação

## Download

Baixe o instalador oficial pela página de Releases:

```text
https://github.com/lstechinfoti/TNavigator-Launcher/releases/latest
```

## Passo a passo

1. Baixe `TNavigatorLauncher_Setup_3.0.0.exe` para instalar pelo assistente.
2. Opcionalmente, baixe `TnavigatorLauncher3_3.0.0.zip` se quiser a versão compilada/manual.
3. Para instalação normal, execute o instalador `.exe`.
4. Conclua a instalação.
5. Abra o **TNavigator Launcher**.
6. Use a tela **Instalação** para instalar, atualizar ou verificar o TNavigator.

## Requisitos

- Windows 10 ou Windows 11
- Arquitetura x64 recomendada
- Permissão de administrador para certificado e dependências
- Conexão com internet

## Observação

Para operações envolvendo certificado, dependências e MSIX, recomenda-se executar o launcher como administrador.
