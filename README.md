# TNavigator Launcher

Página oficial de download do **TNavigator Launcher 3.0**.

O TNavigator Launcher é um instalador, atualizador, gerenciador de backup e ferramenta de diagnóstico para Windows 10/11.

## Recursos

- Instalação e atualização do TNavigator
- Validação obrigatória SHA256
- Verificação de dependências Microsoft VCLibs
- Instalação e remoção de certificado
- Backup e restauração de dados locais
- Logs com filtros
- Tela de diagnóstico
- Instalador profissional para Windows

## Requisitos

- Windows 10 ou Windows 11
- Arquitetura x64 recomendada
- Permissão de administrador para instalação de certificado

## Segurança

Antes de instalar, confira o SHA256 disponível junto com a release.
